从旁指点

实质是什么

东方神功实质是什么？
      是认证机构专用的、武侠式的、现代高级功能体系

什么叫作「现代高级功能体系」？
      不是随便做点高级，随意零零散散的大杂烩、乱炖
      而是一套「高级的方法」，更是一套「高级的标准」
      凭此应对以高网速为代表的现代环境下的系统研发

简单理解

      你如果学会了东方神功，就学会了一整套高级模组
      这些高级模组是成体系的，比拍脑袋乱想要好得多

      学会之后再找麦修行上系统，你自己就懂高品质了
      你不只是说这个表、那个表，相互的关系这般平平
      你还会探讨如何九剑、飞针，怎么样莫言那般精致

你如果没有个「高级的依凭」，你找开发商上系统，说不出几个高级，难成体系，那做出来就素味平平了
学「东方神功」，最起码知道了原来还可以这样子，你看到过好的，就知道样了，那你就会提高级要求了

鞋样子

忆往昔，你的祖辈，做双鞋，在纳鞋底和缝鞋面前，先要找个“鞋样子”。如果没有，那做出来的鞋，肯定就不合脚
今天你，血脉传承，上系统，在谈业务和说逻辑前，也要找个“鞋样子”。如果没有，那做出来的系统，也就好不了

祖祖辈辈都有这个智慧、都不会这么做：没“鞋样子”，弄个鞋底找块布，这个面那个面，咔咔一顿缝，那肯定不行
祖辈智慧，你血脉里肯定有！东方神功，就是你上系统要找的“鞋样子”，按祖辈教我们的智慧上系统，这事就靠谱

东方神功就是高级体系，就是“鞋样子”，祖辈智慧就是让你：先要找个“高级样子”，然后照着样子“弄高级的系统”

麦修行的灵感解释

同一个电影、同一个片段、同一个武功，不同的人脑回路不一样，在理解上会有些不一样
为了让你更懂我是怎么想的，更懂我的设计，所以我对我当时的脑回路，给出些必要解释

独孤九剑 => 高维 => 九剑（扶摇九剑） @麦修行脑回路：
      独孤九剑之所以厉害，就是因为剑法维度高
      所谓「剑法维度」，就是立体、各种角度的
      自上而下攻击，横下旋转冲击，各种角度高速切入，相对于一般平地剑法就是高维
      引申到系统的高级功能，就是跳出一行一行又一行的局限，实现高维度的数据显示

葵花宝典 => 上色 => 飞针（飞针宝典） @麦修行脑回路：
      独孤九剑是高维攻击，教主飞针是专攻一点
      二者都很厉害，是不是可以互补，结合起来
      如果合二为一，让飞针成为九剑的重要修饰
      一个人：既会九剑、又会飞针，岂不是无敌

营救行动 => 灵动 => 莫言 @麦修行脑回路：
      来自「字母剑」的灵感 => 莫言:子剑东流（子表）
      来自「自上而下 抖剑 破 曹少卿的碎刀」的灵感 => 莫言:雕栏破碎（对多）
      来自「踩石进场」和「踩敌离场」的灵感 => 莫言:穿云纵/穿云再纵（秒切换）

教主轻功 => 驾驭 => 神驭 @麦修行脑回路：
      我感觉：教主对自己的轻功，有着神一样的自信，有那种神之驾驭的霸气
      我要完美实现的文武表理论，需要使用者能够灵活调整表格，要神之驾驭
      我认为神驭这个功能非常牛！一般人都做不出，所以神之驾驭、就是霸气

吸星大法 => 获得 => 归宗（百川归宗） @麦修行脑回路：
      按照映射理论，这个很好理解：就是把系统里的数据（内力），变成你自己的（本地Excel）
      百川归宗：百是虚指，意为很多，很多表格（川）归使用者（宗），引申意思就是导出Excel
            一叶归宗：一页（叶）归使用者（宗）
            一川归宗：一个表格（川）归使用者（宗）
            集剑归宗：集合多个表格（剑）归使用者（宗）
            无相归宗：无相就是没有实相（佛教术语）的意思，什么都不用你做，（页/表格/多个表格）归使用者（宗）

凌波微步 => 万行分页 => 万行（御风万行） @麦修行脑回路：
      看每页数据很多的表格，按照映射理论，山川大、河流长，看这种大山大河当然得用轻功了
      所以我给Ta起的名字叫：御风万行xíng，同时这个功能是每页多少行háng，所以也叫万行háng
      用凌波微步跑大山大河，这是一种意境，说不太清楚，这个需要你自己慢慢的“get到那个点”

拈花指 => 智能分页 => 幻叶（幻叶指） @麦修行脑回路：
      如果你实际使用过这个功能、或者看视频去感受，Ta真的就像一种指法，起个名字：幻叶指
      常用这个功能或许能感受到，反正是的，说不太清楚，这个需要你自己慢慢的“get到那个点”

黑木崖的电梯 => 搜素约定 => 密语（陈桥密语） @麦修行脑回路：
      我看到黑木崖的电梯就觉得这里面有密语，就是约定好的，你如果说的不对，就不拉你上去
      引申到系统的高级功能，就是搜素约定、约定好的搜索规则，你用对了就能得到想要的结果
      最初就叫「密语」，至于为什么变成「陈桥密语」，那是因为《东方儿女情》剧情带过去的

六脉神剑 => 颜色标签 => 六彩（六彩神剑） @麦修行脑回路：
      这是我结合「六脉神剑」+「MacBook的Finder标签」+「工作时的需要」产生灵感而设计的

郭靖射雕 => 精致表单 => 箭术（林家箭术） @麦修行脑回路：
      剑法对应的是表格、掌法对应的是信息，想来想去，箭术正好对应表单：向服务器发射数据
      然后想到郭靖射雕，以及《东方儿女情》人物设定，变成「林家箭术」，就有了一箭双雕等

九阴白骨爪 => 快捷键 => 神意（神意夺心爪） @麦修行脑回路：
      要做快捷键 => 武功用手的 => 九阴白骨爪
      九阴白骨爪 => 抠脑袋 => 死
            => 为什么厉害？抠脑袋了，是要害
            => 和脑袋一样重要的要害是：心脏
      综上可得：
            九阴白骨爪 -> 抠脑袋 -> 死
            __夺心爪 -> 抓心脏 -> 死
      然后我通过对键盘的观察，发现 夺(D)心(X)爪(Z) 和我要用的主体区域一致，当时是只缺失 S、E
      教主附身，霸气感觉一来，「S、E」 就变成 「神意」 了
      于是就是：
            神意夺心爪 -> 抓心脏 -> 死
      后来「莫言」里面需要加个【「踩石进场」和「踩敌离场」】，一种「秒切换」的东西，还经常用
            观察下键盘，快捷键方便按的，剩下的可行键都不多了，终定为 C 键盘
                  于是，「秒切换」的东西，就叫「穿（Chuān）云纵」了
      神意夺心爪 也叫 神意夺心超爪 （SEDXCZ 这六个 定为 主键区）
            神(S)意(E)夺(D)心(X)超(C)爪(Z)
            神意夺心超爪 可以这样理解： 神意（教主附身） 夺心（要抓心脏） 超爪（超级厉害的爪）

服部千军 => 数据历史 => 千依（千依子剑） @麦修行脑回路：
      「千条历史记录，以子表的形式、依主数据而存」，依据这个想了个扶桑名字「服部千依」

八音穿心 => 声音 => 八音（八音圣心） @麦修行脑回路：
      「八音穿心」是用声音打别人的，「八音圣心」是听声音自己舒心的

参悟太极 => 主题 => 道意（道意璀粲） @麦修行脑回路：
      放柴奔新：这个是我看电影总结的成语，电影台词：“放下负担，奔向新生命”，意思是：放下贪心柴，奔向新生命！
      这是一种人生智慧，柴代表不该执着的，放下负担才能得到真正好的。执着于你不该拿的，就会错过更美好的东西！
      认证机构上系统的放柴奔新：放下贪心柴（自己搞高级/掌握源码）奔向新生命（拥抱云端，拥抱富信息、大吞吐）

                  《道意璀粲》
            放柴奔新，渐入佳境。
            一二三四，云淡风轻。
      逐句解释
            道意璀粲：道意就是得蓝白大道的意思、璀粲就是又多又好看的意思
            放柴奔新：放下贪心柴（复杂设计）奔向新生命（一二三四简单设计）
            一二三四：简单设计
                  一行：预设
                  二行：字体
                  三行：灰色
                  四行：蓝色
            云淡风轻：简单的设计，就可以实现「微风轻拂、蓝天白云」的感觉

第一步：先做基本盘

把你的业务逻辑，一点点讲给麦修行听，最好是类比着QEO、CCC，这样麦修行容易快速的理解，先实现个基本盘

这个基本盘的原则：
      忽略：九剑、飞针、莫言、无相归宗，就是先不搞高维、不搞上色、不搞灵动、不搞武转文
      所有表格都对（不缺少、不多余），所有表单也都对
      流程逻辑都能走通

这个基本盘的实现：
      麦修行把业务逻辑转为设计
      所有表格都是AI(人谋神力)
      所有表单都是AI(人谋神力)

核心要旨就是：
      忽略「神不能包」的一切，麦修行的工作是把你的业务逻辑转化为设计，利用「神包」，快速实现个基本盘
      这个过程麦修行几乎是不写一行代码的，只做设计转化工作，这么做目的是为第二步的高级化做好必要铺垫

这个基本盘预想的样子：
      Ta也很高级：这些表格只不过没有九剑、飞针、莫言、无相归宗，其余的高级是有的
      Ta是制式的：由于表格、表单都是神包，制式的意思：整整齐齐、品质上是非常好的
      你的工作是：确定流程都对、确定表格都对、确定表单都对，整个基本盘没什么问题

你需要明白的一个道理：
      为什么要做基本盘？
      麦修行需要的是把基本盘全部的代码，让麦修行自己弄的那个AI去扫描、会扫很多遍
                  如果没有这个过程，AI就不太会；让麦修行手搓高级那是不现实的
      所以：没有基本盘，你让麦修行直接实现九剑之高维、莫言之灵动，实在是强人所难
      麦修行是不可能给你直接手写高维、手写子剑，那里面极其的复杂，复杂套着超复杂
      但如果基本盘十之九稳，AI学会了，那麦修行就有这个高级本事了，便可徐徐高级化

第二步：徐徐高级化

随着实现基本盘，麦修行会对你的业务逻辑更加的清楚，麦修行背后的AI也会对基本盘进行反复扫描，神力俱备

常规三步走向高级：
      一: 麦修行会给出些建议：一般一个业务管理做成一个目录，协商九剑高维用哪几剑，如何上色，如何灵动
      二: 凡多表成一块的地方，一般都要把这些表做成一个目录，协商九剑高维用哪几剑，如何上色，如何灵动
      三: 剩下的是一些基础表，大多数不需要上色、不需要灵动，少数重要的挑出来协商：如何上色，如何灵动

后续生态外溢效应：
      以游戏为例：同样的游戏、同样的按钮、同样的设备，就是有人天赋更高，按的更牛，别人就是按的比你好
      东方神功的运用也是如此，只有九种剑式的扶摇九剑，就能用的千变万化，无穷无尽，更何况五绝错综复杂
      麦修行是发明了东方神功，掌控了「AI->东方神功」，但这种厉害是在于AI技术上的，而不是在具体的运用

      真正实际的运用东方神功，更厉害的人是在认证机构，就是说有些人可能会弄的更牛，所以要有借鉴的机制
