若是刷对标 自然神筐妙
念咒扣地翻 变出大方猫
  
神筐变猫，联想记忆：

刷对标：[C]+S，D，E
神筐妙：[C]+shift+S，D，E
大方猫：[C]+J，方向键翻页
按下作用说明
[C]+S刷新刷对标，不是刷新网页，而是刷新数据。
[C]+D对多刷对标，对多模式是否开启，连按切换。
[C]+E标尺刷对标，标尺模式是否开启，连按切换。像 Excel 一样，横向标尺为字母，纵向标尺为数字。
[C]+shift+S神驭神筐妙，进入神驭神的状态：蓝白相间。
[C]+shift+D框选神筐妙，框选模式是否开启，连按切换。
[C]+shift+E妙得神筐妙，下载 Excel 文件，所见即得。
[C]+J大行大方猫，大行模式，加[Jiā]大显示，连按切换。
[C]+⇧大行大方猫，方向键翻页，第一页。
[C]+⇨大行大方猫，方向键翻页，下一页。
[C]+⇦大行大方猫，方向键翻页，上一页。
[C]+⇩大行大方猫，方向键翻页，最后一页。
[C]: Control (Win系统) 或者 Command (Mac系统)
